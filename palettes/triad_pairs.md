# Triad Pairs

**Version 1.0**  
**Harmonic palette for composition engines**

---

## Purpose

Triad pairs provide modern jazz harmonic colour and voice-leading options. Two triads (usually from same scale or related) generate chord voicings and melodic material. Satisfies Rule 6 (Harmonic Depth).

---

## Common Pairings

### Diatonic Pairs (from same scale)

- **C major scale:** C + Dm, C + Em, Dm + Em, etc.
- **D Dorian:** Dm + Em, Dm + C, Em + F
- **G Mixolydian:** G + Am, G + F, Am + Bm

### Lydian Pairs

- **F Lydian:** F + G, F + A, G + A
- Character: floating, #11 colour

### Upper-Structure Pairs

- Triads as upper structure over bass
- Example: D/C = D triad over C bass

### Bitonal Pairs

- Triads from different keys
- Example: C major + F# major (tritone relation)

---

## Application

- Use for chord voicings (stack or alternate)
- Use for melodic material (arpeggiate both)
- Use for reharmonization
- Avoid static repetition; move between pairs
