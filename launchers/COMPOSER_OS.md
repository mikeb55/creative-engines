# Composer OS (V2) — launcher note

**Composer OS** desktop and web apps live under:

- `apps/composer-os-desktop/` — Electron product (single window)
- `apps/composer-os-app/` — web UI
- `engines/composer-os-v2/` — engine and app API

This is **not** the legacy Composer Studio stack. Do not use `composer_studio` or Composer Studio launchers for Composer OS.
