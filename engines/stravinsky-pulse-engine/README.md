# Stravinsky Pulse Engine

Pulse-cell construction, accent displacement, block-based form contrast. Rhythmic architecture first, harmonic support secondary.

## Distinct from Other Engines

- **Monk / Barry Harris:** No swing, no bebop. Dry, block-based.
- **Wayne Shorter:** No lyrical floating. Stark sectional energy.
- **Scofield Holland:** No groove pocket. Pulse and accent displacement.

## Shared Rhythm/Disruption Layer

Uses `shared_rhythm_disruption` for:
- `build_pulse_cell`, `score_pulse_identity`
- `displace_accents`, `score_accent_instability`
- `build_block_contrast_plan`, `score_block_contrast`
- `build_asymmetrical_cycle`, `score_cycle_irregularity`

## When to Use in Hybrids

- Block contrast with Monk harmony
- Pulse arc with Bartok angularity
- Asymmetrical ostinato with Frisell atmosphere
