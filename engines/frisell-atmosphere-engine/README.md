# Frisell Atmosphere Engine

Models Bill Frisell–like open harmony, Americana/ambient spaciousness, pedal tones, and warm ambiguity.

## What This Engine Does

The Frisell Atmosphere Engine generates:

- **Open harmony** — Sus4, maj7, open fifths
- **Americana / ambient spaciousness** — Slow-moving harmonic fields
- **Pedal tones** — Sustained bass/center
- **Melodic afterglow** — Sparse lyric-like motifs
- **Warm ambiguity** — Nonfunctional but accessible
- **Asymmetric phrase flow** — Chamber/ECM atmosphere

## How Atmosphere / Openness Is Modeled

- **Interval profiles** — open_fifths, pedal_melody, wistful_leaps, folk_shadow, ambient_fourths
- **Harmonic profiles** — pedal_field, americana_open, floating_major_minor, suspended_plain, ambient_modal
- **Motif operations** — echo, registral_drift, elongation, silence_spacing, contour_softening, harmonic_shadow

## When to Use in Hybrid Compositions

- **Harmony layer** — Frisell harmony under Shorter or Wheeler melody
- **Atmosphere contrast** — Frisell sections between denser material
- **Pedal texture** — Pair with Bartok or Hill for angular + spacious
- **Americana blend** — Frisell + Wheeler for chamber ballad feel
