# Big Band Bridge

Combines **Wayne Shorter Form Engine** (form plan), **Ligeti Texture Engine** (texture plan), and **Big Band Engine** (large-ensemble host) to produce texture-informed big-band blueprints.

## How Form, Texture, and Big Band Architecture Combine

1. **Form** (Wayne Shorter) — Section order, phrase lengths, asymmetry
2. **Texture** (Ligeti) — Density curve, texture plan, cluster field
3. **Ensemble** (Big Band) — Sectional roles, brass/sax/rhythm, shout capability

The bridge merges these into a single Big Band IR with:
- Form-driven section order and phrase lengths
- Texture-driven density plan and sax texture plan
- Big Band sectional architecture (lead, sax, brass, rhythm)

## Suggested Workflows

1. **Form-led** — Start with Shorter form; add Ligeti texture for density; host in Big Band
2. **Texture-led** — Start with Ligeti texture; map to Shorter form sections; host in Big Band
3. **Ensemble-led** — Start with Big Band; inject Shorter form + Ligeti texture
