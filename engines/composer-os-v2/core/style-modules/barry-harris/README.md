# Barry Harris Style Module

First style module for Composer OS. Emphasises movement over static chords, guide-tone emphasis (3rd/7th), stepwise voice-leading, and dominant gravity. Light 6th↔diminished passing motion.

## Validation

- Voice-leading jumps ≤ 12 semitones
- At least 3 distinct chords (movement, not static)
- No excessive harmonic stasis
