# Scofield Holland Engine

Groove-led modern jazz composition engine. Models chromatic riff logic, blues/funk hybrid language, bass-aware phrase motion, and small-group interplay.

## What It Does

- **Chromatic groove motifs** — semitone approach motion, fourths/tritones in short bursts
- **Blues-shadowed harmony** — funk-blues, chromatic dominant, bass-axis motion
- **Riff-based thematic writing** — motivic rhythmic repetition with variation
- **Bass-aware phrasing** — contour logic that responds to low-register motion
- **Pocket-based asymmetry** — playable small-group forms (4–6, 6–8, 7–9 bar phrases)

## How It Differs

| Engine | Focus |
|--------|-------|
| Monk | Rhythmic displacement, sparse angularity |
| Barry Harris | Bebop enclosure, diatonic voice-leading |
| Wayne Shorter | Modal lyricism, wide intervals |
| Scofield Holland | Groove-first, chromatic riff, blues/funk hybrid |

## When to Use in Hybrids

- **Melody:** Scofield Holland for groove-led heads, riff themes
- **Harmony:** Pairs well with Monk (rhythm), Barry Harris (bebop contrast)
- **Counterline:** Strong with Wheeler Lyric or Frisell for texture contrast
