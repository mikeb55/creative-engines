# Zappa Disruption Engine

Abrupt formal cuts, interruption logic, genre-collision block switching. Jagged rhythmic disruption, asymmetrical cycle breaking.

## Distinct from Other Engines

- **Stravinsky Pulse:** More abrupt, satirical. Interruption over pulse.
- **Monk / Barry Harris:** Anti-bebop. Anti-predictable.
- **Frisell:** Unstable contrast, not atmospheric.

## Shared Rhythm/Disruption Layer

Uses `shared_rhythm_disruption` for:
- `build_interruption_pattern`, `score_disruption_energy`
- `build_block_contrast_plan`, `score_block_contrast`
- `build_asymmetrical_cycle`, `score_cycle_irregularity`

## When to Use in Hybrids

- Interruption collage with Monk blocks
- False return with Wayne Shorter contrast
- Abrupt modal cut with Bartok angularity
