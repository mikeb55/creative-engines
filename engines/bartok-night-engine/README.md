# Bartók Night Engine

Models Bartók "night music" characteristics: sparse textures, atmospheric writing, dissonant clusters, isolated melodic fragments, irregular rhythms, and asymmetric phrasing.

## What Bartók Night Music Is

Bartók's "night music" (e.g. in *Out of Doors*, *Music for Strings*, *Concerto for Orchestra*) features:

- **Sparse textures** — Space between events
- **Atmospheric writing** — Slow, evolving sections
- **Dissonant clusters** — Minor seconds, tritones
- **Isolated fragments** — Short gestures separated by silence
- **Irregular rhythms** — Asymmetric phrase lengths
- **Naturalistic gestures** — Bird calls, insect-like figures
- **Modal + chromatic mixture** — Axis-like tonal centers
- **Asymmetric phrasing** — 3–5, 5–7, 7–9 bar groups

## When to Use in Hybrid Compositions

- **Texture layer** — Bartók night as harmony or counter over Shorter/Hill melody
- **Atmospheric contrast** — Night sections between denser material
- **Cluster harmony** — Pair with Monk or Hill melody for angular + dark
- **Sparse counterline** — Night fragments under Harris or Shorter
