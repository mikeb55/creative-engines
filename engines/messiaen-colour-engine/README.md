# Messiaen Colour Engine

Harmonic colour as primary compositional force. Modes of limited transposition, birdsong-like fragments, non-retrogradable / additive rhythm tendencies, ecstatic suspended harmonic atmosphere.

## Distinct from Other Engines

- **Bartok:** Colour over folk. Static harmonic centres, not developmental.
- **Wheeler:** Lyric over colour. Ecstatic radiance, not song form.
- **Frisell:** Colour over atmosphere. Registral brilliance, not ambient drift.
- **Slonimsky:** Modes over permutation. Harmonic colour, not systematic series.

## When to Use in Hybrids

- Colour panels with Frisell atmosphere
- Birdsong fragments with Wheeler lyric
- Mode-2 harmony with Bartok angularity
- Ecstatic arc with Stravinsky pulse
