# LOCK — Swing / Feel Module

**Activation:** Only when stylistically relevant (swing, groove-led jazz, jazz-pop, ECM-like phrasing).

---

## Purpose

Evaluate and refine rhythmic placement so lines feel played, not merely notated. Reject only obviously square, grid-locked, beat-1-heavy writing when swing feel is in play.

---

## Core Guidance

- **Favour:** Mixed beat placement, syncopation, anticipations, displacement, rests, rhythmic breathing
- **Avoid:** Persistent beat-1 rigidity, quantized-sounding phrasing, barline-heavy uniformity
- **Assess:** Does the line feel played? Would a performer naturally phrase it with authority?

---

## Scope

Broad enough for:
- Straight-ahead jazz
- Modern jazz
- Jazz-pop
- ECM-like sustained phrasing
- Groove-informed melodic writing

---

## Implementation

- This is an **evaluative lens**, not a rigid universal constraint.
- Activates only when the idiom calls for swing/groove-led phrasing.
- Preserves musical judgment; does not force formula.
