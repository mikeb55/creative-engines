# Composition Structure Rules

**Version 1.0**  
**Universal — applies to all styles, ensembles, projects**

---

## Rule 3 — Structural Density Arc (Mandatory)

**Every composition must demonstrate a density arc:**

```
low density → development → peak density → release
```

---

## Rule 5 — Section Clarity (Mandatory)

**All scores must include:**

- Boxed rehearsal letters
- Section boundaries
- Clear structural transitions

---

## Rule 6 — Harmonic Depth (Mandatory)

**Harmony must use one or more of:**

- Tonality Vault
- Triad pair systems
- Polychords
- Interval cycles
- Modal palettes

**Avoid** static harmonic environments unless intentional (e.g. Bartók Night static field).

---

## Structural Requirements

- At least one meaningful contrast point
- Section contrast from texture, register, density, harmony, or role change
- Phrase lengths must not be mechanically symmetrical (prefer 5, 7, 9 bars)
- Each section must contribute a distinct function

---

## Rehearsal Letter Convention

- **A** — Introduction or main statement
- **B** — Development or contrast
- **C** — Climax or expansion
- **A'** — Transformed return
- **Coda** — Closing

Adapt to engine and form. Not every piece uses all letters.
